# BIBLIOTECA DE FONTES 

> **📖 Como usar esta biblioteca**
>
>1. Este é um ficheiro **partilhado por todo o grupo**. Não crie versões locais ou ficheiros separados, todas as fontes devem ser registadas aqui.
>2. **Cada fonte de dados investigada = uma linha nova** nesta tabela. Acrescente a linha **por baixo das que já existem**.
>3. Preencha **todas** as colunas. Se uma coluna não se aplicar à fonte, escreva `Não se aplica`.
>4. Na coluna **Nome**, use o nome oficial da fonte (ex.: `INE – Estatísticas do Comércio Internacional`). Evite siglas sozinhas.
>5. Na coluna **Link direto**, coloque o URL que leva diretamente à página de descrição ou de download dos dados. Evite links genéricos (ex.: página inicial do INE).
>6. A coluna **Período coberto** deve indicar o intervalo de datas (ex.: `2010-2024` ou `desde 2000`). 
>7. Na coluna **Frequência de atualização**, use termos como `anual`, `mensal`, `trimestral`, `diária`, `sem atualização prevista` ou `Não informada`.
>8. Para colunas com **múltiplos itens** (ex.: `Dados disponibilizados`, `Formato de export`, `Limitações conhecidas`), separe os itens com `<br>` (ex.: `Excel<br>CSV`). Isto evita que a tabela Markdown se parta.
>9. A coluna **Responsável** deve conter o **nome do estagiário** que investigou e registou a fonte. Use o formato `Nome Apelido`. Se a fonte foi registada a partir de um comentário de outra pessoa, mantenha o nome original de quem fez a investigação.
>10. **Faça commit** assim que adicionar uma linha (ou um pequeno grupo de linhas). Mensagem de commit recomendada: `docs: adiciona fonte [Nome da fonte] à biblioteca`. Isto mantém o histórico claro e evita conflitos.
>11. Se descobrir que uma fonte **já existe** na tabela (mesmo com nome ligeiramente diferente), **não adicione uma linha duplicada**. Em vez disso, atualize a linha existente (clicando em “Edit” no GitHub) e acrescente a nova informação ou corrija a existente.
>12. **Nunca apague linhas** sem autorização do coordenador. Se uma fonte deixar de ser relevante, simplesmente acrescente `(descontinuada)` ao nome na coluna **Nome**.
>13. Antes de adicionar uma fonte, verifique rapidamente se ela **já está registada** – use a pesquisa do GitHub (Ctrl+F) ou olhe a tabela. Evitar duplicados é responsabilidade de todos.
