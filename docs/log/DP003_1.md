# Log de Coleta - DP003.1

> **Instruções:**
> 1. Uma linha por ficheiro descarregado nesta subtarefa.
> 2. **Não edite o `log.md` diretamente.** Este ficheiro é consolidado pela líder.
> 3. Faça commit assim que acrescentar uma linha.
> 4. Mensagem de commit sugerida: `docs(DP003.1): regista coleta [nome do ficheiro]`
>
> **Notas por coluna:**
> 1. **Ficheiro:** nome do arquivo como link para onde está no GitHub, no formato `nome.csv` (ex: `[nome.csv](link-do-github)`).
> 2. **Data de coleta:** dia em que fez o download (formato AAAA-MM-DD), não a data do relatório original.
> 3. **Transformações** e **Limitações** com vários itens: separe com `<br>` para não partir a tabela.
> 4. **Responsável:** nome completo + link GitHub (ex: `Nome Apelido`).
> 5. Se uma coluna não se aplicar, escreva `Não aplicável` ou `Nenhuma`. Nunca deixe em branco.
